# aquarium-server
基于SpringBoot+Vue前后端分离的生态水族馆网站系统，系统包含管理员、员工、会员角色，功能如下： 管理员：首页统计（评分统计、门票销售额统计）、用户管理、讲解管理、门票管理、门票订单、项目管理、项目预约、讲解预约、讲解评价、系统留言、公告信息、关于本馆、个人信息密码修改。 讲解员：讲解预约（确认完成）、讲解评价（回复）、公告查看、个人信息密码、密码修改。 用户：注册、登录、首页、门票购买（查看水族馆介绍、查看门票、购票）、娱乐项目（项目信息、详情、预约）、讲解员（讲解员信息、详情、预约、评价信息）、系统留言、系统公告。用户还具有个人中心功能，功能包括：门票购买、项目预约、讲解预约、讲解评价、系统留言、个人信息、密码修改。
